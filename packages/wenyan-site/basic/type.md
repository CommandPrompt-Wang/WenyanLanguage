## 爻

吾有一恒爻。性阴。

此所谓「假」。

另有一恒阳爻。

名之曰「真」。

```ts
const 假: bool = false;
const 真: bool = true;
```


## 言

吾有一言。曰「「噫吁戏」」。

名之曰「蜀道难」。

又有一言。名之曰 「粗鄙之语」 。

「「请诸位静听」」。

「「天数有变，神器更易」」

```ts
let 蜀道难: string = "噫吁戏";
let 粗鄙之语: string = "请诸位静听\n天数有变，神器更易";
```

## 数

吾有一数。曰三。名之曰「甲」。

有数五十。名之曰「大衍」。

```ts
let 甲: number = 3;
let 大衍: number = 50;
```

吾有三数。

曰一。曰三。曰五。

名之曰「甲」曰「乙」曰「丙」。

```ts
let 甲=1,乙=3,丙=5;
```


## 列

吾有一列。名之曰「列」。

吾有一数列。名之曰「斐波那契」。

```ts
let 列: any[] = [];
let 斐波那契: number[] = [];
```
