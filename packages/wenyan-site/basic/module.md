吾尝观「「算经」」之书。方悟「正弦」「余弦」「圆周率」之义。

```ts
import {正弦, 余弦, 圆周率} from '算经'
```

有一书名曰 《九章算数》, 载 「求和」之术, 今名之曰 「加」


```ts
import { 求和 as 加 } from 九章算数 
```
